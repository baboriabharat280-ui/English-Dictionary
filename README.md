# 📖 English Dictionary

A simple and responsive **English Dictionary** web application built using **HTML, CSS, and JavaScript**. The application allows users to search for English words and instantly view their meanings, pronunciation, parts of speech, and example usage through a dictionary API.

## ✨ Features

* 🔍 Search for English words
* 📚 View word meanings and definitions
* 🔊 Listen to pronunciation (if available)
* 📝 Display parts of speech and examples
* 📱 Responsive and user-friendly interface
* ⚡ Fast API-based search

## 🛠️ Technologies Used

* HTML5
* CSS3
* JavaScript (Vanilla JS)
* Dictionary API

## 🚀 How to Use

1. Open the application in your browser.
2. Enter an English word in the search box.
3. Press **Search** or hit **Enter**.
4. View the word's definition, pronunciation, and additional details.
